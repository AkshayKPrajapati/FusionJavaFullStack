Q. What is a database?
Answer . A database is an organized collection of data that is stored and managed so it can be easily accessed, updated, and retrieved.
Q. Where is data stored in a database?
Answer . Data in a database is stored in a structured way, usually like this:

Tables → Main place where data is stored
Rows (records) → Each row represents one item (e.g., one student)
Columns (fields) → Each column represents a type of information (e.g., name, age)
Example:
A Student table might look like:
| ID | Name | Age |
| -- | ---- | --- |
| 1  | Ravi | 20  |
| 2  | Anya | 21  |


SQL commands are commonly grouped into 5 categories: 
1.DDL (Data Definition Language) 
2.DML (Data Manipulation Language) 
3.DQL (Data Query Language) 
4.DCL (Data Control Language) 
5.TCL (Transaction Control Language) 

1. DDL (Data Definition Language): Used to define or modify the structure of database objects.
CREATE,ALTER,DROP,TRUNCATE,RENAME
👉 Example:
CREATE TABLE students (
 id INT,
 name VARCHAR(50)
);

2. DML (Data Manipulation Language): Used to manipulate data inside tables.
Commands:
INSERT
UPDATE
DELETE
👉 Example:
INSERT INTO students VALUES (1, 'Rahul');

3. DQL (Data Query Language): Used to retrieve data from the database.
Command:
SELECT
👉 Example:
SELECT * FROM students;

4. DCL (Data Control Language) :Used to control access/permissions.
Commands:
GRANT
REVOKE
👉 Example:
GRANT SELECT ON students TO user1;

5. TCL (Transaction Control Language) :Used to manage transactions in the database.
Commands:
COMMIT
ROLLBACK
SAVEPOINT
👉 Example:
COMMIT;
